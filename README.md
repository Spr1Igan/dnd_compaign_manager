# D&D Campaign Manager

**Текущая версия:** `0.7.0`

D&D Campaign Manager — веб-приложение на Laravel для ведения персонажей Dungeons & Dragons 5e. Сейчас основной фокус проекта — удобный лист персонажа: создание, просмотр, редактирование, автоматические расчёты и быстрые изменения игровых значений во время сессии.

## Что уже реализовано

- Регистрация, вход и выход пользователя.
- Главная страница приложения и навигация.
- Список персонажей текущего пользователя.
- Создание, просмотр, редактирование и удаление персонажа.
- Отдельные маршруты персонажей в `routes/characters.php`.
- Стилизованная форма персонажа в духе бумажного character sheet.
- Русскоязычный интерфейс листа персонажа.
- Справочники рас, классов, предысторий, навыков и языков через сидеры.
- Автоподтягивание данных от выбранной расы, класса и предыстории.
- Расчёт итоговых характеристик с учётом расовых бонусов.
- Расчёт модификаторов характеристик.
- Расчёт стартовых HP от кости хитов класса и Телосложения.
- Расчёт КД без доспеха как `10 + модификатор Ловкости`.
- Поддержка ручного КД для доспехов, щитов и особых правил.
- Расчёт бонуса мастерства по уровню.
- Расчёт спасбросков с учётом владения класса.
- Интерактивное изменение текущих HP на странице персонажа.
- Интерактивное прибавление и вычитание опыта.
- Подсказки по характеристикам, навыкам, спасброскам, КД и бонусу мастерства.
- Кастомное подтверждение удаления персонажа.
- Тесты для ключевой логики КД.

## Версионирование

Проект использует формат `0.x.y`.

- `0` — проект ещё находится в активной разработке до первой стабильной версии `1.0.0`.
- `x` — добавление новой функции или заметного пользовательского сценария.
- `y` — мелкие исправления, визуальные правки, уточнение расчётов, документация без изменения основной логики.

Примеры:

- `0.4.0` — добавлена новая функция.
- `0.4.1` — исправлена ошибка или улучшена мелкая деталь в рамках версии `0.4`.
- `0.5.0` — добавлен следующий крупный блок функциональности.

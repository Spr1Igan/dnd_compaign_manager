# D&D 5e — собственная база сокровищ

Состав:
- coins.json — валюты;
- gems.json — самоцветы по ценовым категориям;
- art_objects.json — предметы искусства;
- jewelry.json — украшения;
- relics.json — реликвии с сюжетными крючками;
- books.json — редкие книги и рукописи;
- mundane_loot.json — обычный интересный лут;
- rare_goods.json — редкие торговые товары;
- treasure_tables.json — собственные таблицы индивидуальных сокровищ и кладов;
- generator_schema.json — рекомендации для генератора.

Эта база не воспроизводит таблицы Dungeon Master's Guide и может свободно использоваться в собственном проекте.

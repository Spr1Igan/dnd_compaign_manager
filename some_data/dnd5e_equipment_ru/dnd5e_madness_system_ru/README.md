# Расширенная система безумия для D&D 5e

Файлы:
- rules.json — основные правила применения и накопления эффектов;
- minor_madness.json — 30 лёгких эффектов;
- major_madness.json — 30 тяжёлых эффектов;
- chronic_madness.json — 40 хронических эффектов;
- triggers.json — причины проверок;
- treatments.json — способы лечения;
- tables.json — броски тяжести и эскалации;
- tags.json — теги для фильтрации;
- manifest.json — состав набора.

Всего: 100 эффектов.

Система авторская и не воспроизводит официальные таблицы Dungeon Master's Guide.

# Основные правила ведущего D&D 5e

Набор содержит:
- состояния;
- типы урона;
- чувства;
- размеры существ;
- мировоззрения;
- отдых и истощение;
- путешествия;
- болезни и яды;
- изготовление предметов;
- деятельность между приключениями;
- наём работников;
- экономику;
- пороги опыта для встреч;
- правила оценки сложности встреч;
- таблицу CR, опыта и бонуса мастерства.

По запросу не включены генераторы NPC, квестов, таверн, городов,
подземелий, событий, ловушек и сокровищ, а также погода, освещение,
укрытия и типы перемещения.

Это структурированная русская справочная база, а не копия текста
Dungeon Master's Guide.

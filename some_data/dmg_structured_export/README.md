# Полный структурированный экспорт «Руководства Мастера» D&D 5e (RU)

Содержимое:
- `metadata.json` — метаданные источника и контрольная сумма.
- `toc.json` — оглавление и ссылки на файлы разделов.
- `chapters/*.json` — полный извлечённый текст по страницам, абзацам, строкам и заголовкам.
- `pages.jsonl` — все 322 страницы в одном потоковом формате JSON Lines.
- `tables/*.json` — кандидаты строк таблиц, выделенные по координатам текста.
- `entities/*.json` — предметы, артефакты, ловушки, болезни, яды, безумие, осадное снаряжение и дополнительные правила.
- `indexes/headings.json` — индекс всех распознанных заголовков.

Важно: PDF имеет сложную двухколоночную верстку, декоративные буквицы и таблицы без явных границ. Поэтому экспорт является максимально полным машинным извлечением, но отдельные заголовки и сложные таблицы могут потребовать ручной нормализации перед импортом в БД.
